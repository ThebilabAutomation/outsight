Behavior Index

A métrica que mede a propensão à ação do seu cliente em cada contexto de decisão — revelando a massa submersa do iceberg: motivações, esforços e vieses que os dashboards tradicionais não enxergam.

BI = IM − IE
IM · Motivação = (Desejo + Necessidade + Urgência) / 3
IE · Esforço = (Financeiro + Cognitivo + Físico + Tempo + Emocional) / 5
BI > 0 → ação BI ≈ 0 → ponto crítico BI < 0 → inação
